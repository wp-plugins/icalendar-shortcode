Import iCalendar events in your WordPress articles

== Description ==
This plugin adds an `ics-shortcode` shortcode that allows you to import events from an iCalendar file.

== Installation ==
1. Upload the `ics-shortcode` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add the [ics_events url="http://example.com/calendar.ics" nb="3" locale="fr_FR"] shortcode to page or article
